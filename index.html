Installation

You can import the main Cordelia Component and render it.

1
import React from 'react';
2
import ReactDOM from 'react-dom';
3
import Cordelia from 'cordelia-react';
4
​
5
ReactDOM.render(<Cordelia color="#EEA55B" />,
6
                document.querySelector('#picker'));

You can also add the script file into your html.

<script crossorigin src="https://unpkg.com/react/umd/react.production.min.js"></script>
<script crossorigin src="https://unpkg.com/react-dom/umd/react-dom.production.min.js"></script>
<script src="/node_modules/cordelia-react/dist/js/cordelia.min.js"></script>
<script>
ReactDOM.render(
    React.createElement(Cordelia, {color:'#EEA55B'}),
    document.querySelector('#app')
);
</script>

Put the CSS file in the head tag.

<link rel="stylesheet" href="/node_modules/cordelia-react/dist/css/cordelia.min.css" />

Color Format

  • Value Type : String
  • Default Value : hex

The format of the picker's value. Cordelia supports three types of formats: HEX, RGB and HSL. If the opacity is less than 1, HSL is automatically converted to HSLA, HEX and RGB are automatically converted to RGBA.

Picker Style

  • Value Type : Number
  • Default Value : 0

Cordelia comes with two style options. In the first style, the main box allows setting the saturation and lightness of the color and the second box allows setting the color; this is the default.   • onOpen : Fires when the picker is open.

  • onSave : Fires when the selection is saved.
